## How To Use

First Step: go to path docker-jenkins
```
cd /path/docker-jenkins/
```

Second Setep: start jenkins for docker
```
docker-compose up -d
```

Third Setep: find token in logs docker by command
```
docker-compose logs -f jenkins
```
